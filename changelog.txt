v0.3.1 (2026-06-04)
--------------------------
Improvements:
  - Certification prefix now defaults to the country preset value (e.g. NL: for Netherlands, Rated for US) instead of being empty
  - Added "Override certification prefix" toggle (advanced setting) to use a custom prefix
  - The prefix text field only appears when the override toggle is enabled

v0.3.0 (2026-06-04)
--------------------------
  - Initial release
  - Automatically fills missing age certifications for movies and TV shows
  - 9 country presets: NL, BE, DE, AT, GB, US, FR, DK, SE
  - Searches TMDB, OMDB, and national rating authority websites
  - FSK (Germany), BBFC (UK), Medieraadet (Denmark), Kijkwijzer (Netherlands)
  - Converts between rating systems using conservative rounding
  - Per-scraper enable/disable toggles
  - Runs as a background service with configurable scan intervals
  - Manual scan trigger from Programs menu
  - Retry tracking for items not yet in any database
  - Structured debug logging to separate file
